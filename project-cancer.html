Breast Cancer Detection via CNN

A convolutional neural network trained on medical imaging data to detect breast cancer with 88% accuracy — demonstrating the potential of deep learning in early-stage oncology screening.

Type
Independent Project
Stack
Python, TensorFlow
Accuracy
88% on test set
Data
Kaggle

Overview

Breast cancer is one of the most common cancers globally. Early detection dramatically improves survival rates — but radiological screening is limited by specialist availability. This project explores whether a CNN can provide reliable automated detection from medical imaging data.

Approach

  • Dataset: labeled breast cancer imaging dataset sourced from Kaggle
  • Preprocessing: normalization, augmentation (flips, rotations) to improve generalization
  • Architecture: custom CNN with convolutional blocks, max pooling, and dense classification head
  • Binary output: malignant vs. benign prediction
  • 88% test accuracy with balanced evaluation across both classes

Result

Model Performance

88% accuracy on the held-out test set, with balanced performance across both malignant and benign classes — demonstrating the model generalizes rather than overfitting to class imbalance.
