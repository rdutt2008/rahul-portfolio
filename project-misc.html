Handwritten Digit Recognition (ANN)

Built an Artificial Neural Network (ANN) to recognize handwritten digits from the MNIST-style dataset. Implemented forward propagation, backpropagation, and hyperparameter tuning from scratch, building intuition for the fundamentals before moving to CNN-based architectures.

Cancer Prediction via Logistic Regression

Applied logistic regression to classify cancer samples as malignant or benign using medical feature data. This project focused on feature engineering, sigmoid activation, and evaluating model performance via precision/recall curves.

Home Price Prediction via Linear Regression

Trained a linear regression model to predict home sale prices based on property features. Explored multicollinearity, regularization (Ridge/Lasso), and residual analysis to understand when linear models hold and when they break.

Diet Monitoring App (MIT App Inventor)

Built a mobile app for tracking daily nutritional intake. The app logs meals, computes macro breakdowns, and displays weekly summaries — designed with a clean, accessible UI for everyday use. Built entirely in MIT App Inventor, which introduced mobile development concepts including event-driven programming and persistent local storage.
