Diabetic Retinopathy Early Detection

Early detection of diabetic retinopathy using morphological blood vessel annotation and a CNN classifier — achieving ~70% accuracy on retinal images. 4th Place at Regional Science Fair.

Type
Independent Research Project
Award
4th Place — Regional Science Fair
Stack
TensorFlow, NumPy, Pandas
Accuracy
~70% on test set

The Problem

Diabetic Retinopathy (DR) is one of the leading causes of blindness worldwide. Early detection is critical — but manual review of retinal scans by specialists is expensive and slow. An automated screening tool can flag high-risk patients before vision loss occurs.

Methodology

  • Input: retinal fundus images with varying DR severity levels (Kaggle dataset)
  • Preprocessing: morphological transformations (erosion, dilation, top-hat) to isolate blood vessels
  • Blood vessel maps serve as secondary input — key structural indicators of DR progression
  • CNN model trained on both original and processed vessel-annotated images
  • Binary and multi-class predictions for DR severity

Why Blood Vessel Annotation?

Key Insight

Diabetic Retinopathy manifests as changes in the retinal vasculature — microaneurysms, hemorrhages, and neovascularization. By explicitly highlighting vessels through morphological transforms before CNN classification, the model focuses on the most diagnostically relevant features.

Results

  • ~70% classification accuracy on held-out test set
  • 4th Place finish at Regional Science Fair (Technology category)
  • Demonstrated value of domain-specific preprocessing over raw-image CNN approaches
